# Simple Web3 Wallet

This is a minimal web3 wallet example inspired by the design of OKX Wallet. It uses [ethers.js](https://docs.ethers.org/) to connect to an Ethereum provider like MetaMask.

## Usage

Open `index.html` in a browser that has a web3 provider (e.g., MetaMask). Click the **Connect Wallet** button to authorize the connection and display your address and balance.

The interface features a dark theme reminiscent of OKX Wallet. Feel free to customize the styles in `styles.css` to better match your desired design.
